# RNTK_UCI
This is an efficient GPU implementation of recurrent neural tangent kernel (RNTK) for UCI dataset.

## dependencies
jax

symjax

sklearn

## reproduce results
open a terminal `cd $PROJECT` where `$PROJECT` is the repo location and then simply run `bash main.sh`.
